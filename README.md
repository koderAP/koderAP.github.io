# Anubhav Pandey - Portfolio Website

Modern, responsive portfolio website showcasing my journey as a **System Programmer** and **AI Enthusiast**. Built with cutting-edge web technologies and featuring interactive animations and smooth user experience.

## ✨ Features

### 🎯 Interactive About Section
- **Horizontal Card Slider**: Navigate through three main expertise areas with smooth animations
- **Navigation Controls**: Left/right arrow buttons and indicator dots for easy navigation
- **Auto-sliding**: Cards automatically cycle with pause on hover
- **Touch/Swipe Support**: Mobile-friendly touch navigation
- **Keyboard Navigation**: Arrow keys and number keys (1-3) for accessibility

### 🛠 Technologies Slider
- **Multi-slide Carousel**: Showcases 12 different technology categories across 6 slides
- **Smooth Transitions**: CSS3 animations with cubic-bezier easing
- **Auto-advance**: Automatically cycles through technologies with user controls
- **Responsive Design**: Adapts beautifully to all screen sizes

### 🎨 Modern UI/UX
- **Dark/Light Theme Toggle**: Seamless theme switching with localStorage persistence
- **Smooth Scrolling**: Enhanced navigation experience with scroll animations
- **Responsive Grid Layouts**: Mobile-first design that scales perfectly
- **Interactive Elements**: Hover effects, transitions, and micro-animations
- **Professional Typography**: Inter font family for excellent readability

## 🚀 Technical Implementation

### Frontend Technologies
- **HTML5**: Semantic markup with accessibility features
- **CSS3**: Modern styling with CSS Grid, Flexbox, and animations
- **Vanilla JavaScript**: No framework dependencies, pure performance
- **Font Awesome**: Professional iconography
- **CSS Variables**: Dynamic theming system

### Key Features
- **Intersection Observer API**: Efficient scroll-based animations
- **CSS Grid & Flexbox**: Advanced layout systems
- **CSS Custom Properties**: Dynamic theme switching
- **Touch Events**: Mobile gesture support
- **Local Storage**: Theme preference persistence
- **Semantic HTML**: Screen reader friendly

## 🛠 Installation & Setup

### Quick Start
```bash
# Clone the repository
git clone https://github.com/koderAP/koderAP.github.io.git

# Navigate to project directory
cd koderAP.github.io

# Open in browser (or use live server)
open index.html
```

### Development
No build process required! Simply open `index.html` in your browser or use a live server for development.

### File Structure
```
├── index.html          # Main HTML file
```
