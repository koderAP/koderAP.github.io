Linux Kernel Resource Usage Tracker & Limiter

This project was developed as part of the Operating Systems Course under the guidance of Prof. Smruti Ranjan Sarangi. I modified the Linux Kernel source code to add a resource usage tracker and limiter for processes. The implementation introduces new system calls and kernel modules to monitor and enforce limits on heap memory usage and the number of open files for registered processes.

The project features real-time resource monitoring via /proc/tracker_status, automatic quota enforcement (processes exceeding limits are killed), robust cleanup using kprobes, and compatibility with both x86_64 and ARM64 architectures. The patch is provided for Linux kernel versions 6.1.6 and 6.13.4, with detailed documentation and a user-friendly interface for registration, fetching, and quota management.
